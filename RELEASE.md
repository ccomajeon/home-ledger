## 릴리즈 내역

### v0.2.1 - 2026-07-20
- 신규
  - Argon2 비밀번호 해시와 HTTPOnly 세션을 사용하는 로컬 `SYSTEM` 관리자 로그인 추가
  - 로그인 실패 횟수 제한과 공개 인증 방식 조회 API 추가
- 개선
  - 로그인 화면, 사이드바, 모바일 내비게이션, 재정 대시보드를 최신 데이터 중심 UI로 전면 개편
  - 일괄 빌드가 백엔드·프런트엔드의 포맷, 린트, 테스트, 빌드 실패를 즉시 중단하도록 강화
  - 백엔드 자동화 테스트를 16개로 확대하고 실제 브라우저 데스크톱·모바일 검증 추가
- 오류수정
  - Google OAuth 미설정 환경에서 로그인할 수 없던 문제를 로컬 관리자 인증으로 해결
  - 로컬 비밀 설정과 테스트 임시 파일이 Git 변경 대상으로 노출되지 않도록 제외 규칙 보완

### v0.2.0 - 2026-07-19
- 신규
  - Google OAuth 로그인, 서명 세션 쿠키, 계정·도메인 허용 목록 인증 구현
  - 거래 CRUD·검색·필터·요약과 카테고리·결제수단 관리 API 및 화면 구현
  - OWNER 계정 관리, 감사 로그, SQLite 백업·복원 기능 구현
  - 백엔드 14개·프런트엔드 4개 자동화 테스트와 로컬 서버 스모크 테스트 추가
  - Windows/Linux용 일괄 빌드·실행 스크립트 추가
- 개선
  - 반응형 가계부 UI와 FastAPI의 프로덕션 프런트엔드 단일 서버 제공 방식 도입
  - 프런트엔드 의존성 보안 감사 결과를 0건으로 정리
  - 실행·환경 설정·검증 방법을 최신 구현에 맞게 문서화
- 오류수정
  - 손상된 백엔드·프런트엔드 README 한글 인코딩 복구
  - Windows PowerShell 5에서도 스모크 테스트가 동작하도록 호환성 보완
